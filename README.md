## Getting Started

## Libraries & Frameworks

-   React Typed
-   Embla Carousel
-   Embla Carousel React
-   Embla Carousel React
-   Prettier Plugin (Tailwind)
-   Tailwind CSS
-   Framer Motion

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
